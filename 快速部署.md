# ⚡ 快速部署指南

## 🎯 三步完成部署

### 第一步：上传到GitHub
```bash
# 在项目目录执行
git init
git add .
git commit -m "GPT充值系统"
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

### 第二步：Vercel导入
1. 访问 https://vercel.com
2. 用GitHub登录
3. 点击 "New Project" → 导入你的仓库
4. 点击 "Deploy"

### 第三步：设置环境变量
在Vercel项目设置中添加：
- `SECRET_KEY` = `your-super-secret-key-here`
- `SESSION_TIMEOUT` = `1800`

然后重新部署即可！
